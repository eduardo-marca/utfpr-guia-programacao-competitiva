# O Que é Programação Competitiva

Programação Competitiva é um esporte onde os competidores tentam resolver problemas de programação em um tempo limitado. Esses problemas geralmente envolvem algoritmos e estruturas de dados, e os competidores precisam escrever código eficiente para resolver os problemas corretamente dentro do tempo estipulado.

Ganha o competidor que resolver mais problemas corretamente em menos tempo. A programação competitiva é uma ótima maneira de melhorar suas habilidades de programação, aprender novos algoritmos e se preparar para entrevistas de emprego em tecnologia.

Em uma competição tradicional, cada equipe é composta por até três membros que dividem um mesmo computador para resolver os problemas. Ao resolver um problema, a equipe submete sua solução para um sistema de avaliação automática, que verifica se a solução é correta e eficiente. As equipes recebem pontos por cada problema resolvido corretamente, e o tempo gasto para resolver os problemas também é levado em consideração para desempates.

Também existem competições individuais, onde cada competidor tem seu próprio computador para resolver os problemas. Essas competições são mais comuns em plataformas online, onde os competidores podem participar de qualquer lugar do mundo de forma remota.

## Maratona SBC de Programação

A Maratona SBC de Programação é uma competição anual organizada pela Sociedade Brasileira de Computação (SBC) que reúne estudantes universitários de todo o Brasil para competir em programação. A competição é realizada em várias etapas, começando com uma fase regional e culminando em uma final nacional.

## Maratona ICPC

A Maratona ICPC (International Collegiate Programming Contest) é uma competição internacional de programação para estudantes universitários. A competição é organizada pela Association for Computing Machinery (ACM) e reúne equipes de todo o mundo para competir em programação.

As primeiras etapas da competição são regionais (no Brasil são as Fases Regionais da SBC), onde as equipes competem para se classificar para a final mundial. A final mundial é realizada anualmente em um local diferente e reúne as melhores equipes de todo o mundo para competir por prêmios e reconhecimento.

## Como Começar

A melhor maneira de começar na programação competitiva é praticar resolvendo problemas. Existem várias plataformas online onde você pode encontrar uma grande variedade de problemas para resolver.

Você pode resolver problemas de constests passados, participar de competições ao vivo ou simplesmente praticar com problemas de dificuldade variada.

### Sites de Prática

Os principais sites de prática para programação competitiva incluem:

- [Codeforces](https://codeforces.com/) (Principal plataforma de programação competitiva do mundo).
- [AtCoder](https://atcoder.jp/) (Principal plataforma de competições para iniciantes).
- [LeetCode](https://leetcode.com/) (Principal plataforma de entrevistas técnicas, mas também tem competições regulares).
- [HackerRank](https://www.hackerrank.com/)
- [CSES](https://cses.fi/)
- [CodeChef](https://www.codechef.com/)
- [USACO](http://www.usaco.org/)
- [SPOJ](https://www.spoj.com/)

!!! tip
    O AtCoder realiza competições para inciantes semanalmente, normalmente aos sábados 9h (horário de Brasília), e é uma ótima maneira de começar a competir.
